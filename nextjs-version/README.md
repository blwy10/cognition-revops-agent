# Next.js + TypeScript replica

This directory is a full web-stack replica of the Python/PySide6 RevOps app using:
- Next.js App Router
- React + TypeScript
- Material UI

## Features mirrored
- Data generation + existing JSON loading
- Rule settings and enable/disable toggles
- Run analysis and produce issues
- Previous run history
- Inbox with issue detail, snooze/resolve/reopen
- Local persistence with `localStorage`

## Run
```bash
cd nextjs-version
npm install
npm run dev
```
